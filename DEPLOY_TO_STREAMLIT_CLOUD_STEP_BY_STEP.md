# Step-by-Step: Update GitHub and Streamlit Cloud

## If you already deployed the previous version

1. Unzip this package.
2. Open your GitHub repo.
3. Upload/replace these files from the new folder:
   - app.py
   - README.md
4. Commit changes.
5. Go to Streamlit Cloud.
6. Click Reboot app.

## If this is your first deployment

1. Create a private GitHub repo called derby-v3-demo-summary.
2. Upload everything inside this folder.
3. Go to Streamlit Community Cloud.
4. Create a new app from that repo.
5. Main file path: app.py
6. Deploy.
7. Open the Streamlit URL on your phone.
